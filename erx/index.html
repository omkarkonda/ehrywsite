Golden Thread:

The EHR includes a predefined Treatment Plan web form that can be used to document a patient's treatment plan. EHR can customize master list of Goals, Objectives and Interventions so that the user can pick from them.

Easy Form Permissions:

The system allows setting access permissions that control who can view, edit, and document the form, along with this additional permission can be set up for all the above mentioned features based on Users / Roles.

Form Forwarding:

As mentioned above, a documented form can be manually forwarded to another user for review or viewing. Additionally, automated rules can be set to forward the form to a supervisor for finalization, co-signature, or viewing after it has been documented. The supervisor can also add comments and send it back to the supervisee for completion.
