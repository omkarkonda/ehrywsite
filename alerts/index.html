Streamline Communication with Secure Messaging

Stay connected with patients effortlessly through our secure Inbox & Messages feature. Designed for healthcare teams, it enables seamless, real-time communication, prioritizing patient privacy.

Automated Reminders and Alerts

Create automated reminders for users based on various modules actions. Example: Notes, Documents, Program Admissions, Appointments, Demographics & Portal actions, Authorizations etc..

  1. View all automated reminders for all your clients in a single reminders window
  2. Filter reminders in this window by date
  3. Mark reminders as completed in this window
  4. Export reminder dues from this window
  5. Give selected users or roles access to reminders
  6. Turn reminders off manually or automatically
  7. Block the documentation of notes based on reminder conditions
  8. Release the block automatically once specified conditions are met

  2. Edit and sign off notes
  3. Bulk sign-off draft notes
  4. Annotate your notes with comments
  5. Print/pdf/fax/email notes
